vuex

安装

npm install vuex es-promise –save这个命令来安装vuex和es6-promise，因为vuex依赖Promise。在src文件下面新建store文件，然后在store文件夹下面创建如下文件:

  • action.js，action.js里面是vuex state 异步的相关操作。你也可以在对应的action里面进行一些接口的调用，然后进行数据的存在，但是我个人不是很喜欢。
    import * as types from "./types";
export default{
  [types.TOKEN]({commit}, token){
  commit(types.TOKEN, token);
  },
}
  • getters.js，getters.js是对state的数据进行一些操作，然后被返回。
  • mutation.js, mutation.js实现state的同步保存。
  • store.js， 初始化state的值，创建一个store。
  • types.js state类型变量定义。
    项目中如果组件的传递数据不是很复杂，可以直接用官方推荐的new EventBus()的方式，不需要用到vuex,因为没有什么数据传递，用vuex反而是个累赘。
